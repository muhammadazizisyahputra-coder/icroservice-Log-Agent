# 基础信息录入脚本

## 开场白

```
我来帮你创建这个人的 Skill。

我需要了解几个问题，每个都可以跳过，但至少需要一个昵称。
```

---

## 问题序列

### Q0：她/他是谁？【新增，最优先】

```
这是一个什么样的人？（这决定了我怎么提取信息）

选择一个：
  [1] 前任（ex）
  [2] 最好的朋友 / 闺蜜 / 哥们
  [3] 家人（妈妈、爸爸、哥哥、妹妹等）
  [4] 导师 / 老师 / 教练
  [5] 同事 / 领导
  [6] 其他（自定义）

例：「前任」「最好的朋友」「我妈」"大学老师"
```

根据选择，后续问卷会动态调整。

---

### Q1：昵称/代号

```
她/他怎么称呼？（昵称、小名或代号都行）

例：小美、Lucy、王老师、Tom、妈妈
```

- 接受任意字符串
- slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"小美" → `xiaomei`，"糖糖" → `tangtang`）
- 英文直接小写 `-` 连接（"Sweet Luna" → `sweet-luna`）

---

### Q2：基本信息【动态问卷，根据角色类型调整】

#### 如果是前任：
```
用一句话描述你们的爱情——在一起多久、怎么认识的、分手多久、她的职业

例：在一起三年 大学同学 分手一年 她做设计
```

从回答中解析：
- 在一起时长
- 认识方式
- 分手时长
- 职业

#### 如果是朋友：
```
用一句话描述你们的友谊——怎么认识的、相交多久、她做什么的、相处方式

例：大学同学 相交5年 都在一线城市 她做产品经理 经常一起出去玩
```

#### 如果是家人：
```
用一句话描述你们的关系——她是你什么人、她的年纪/性格、你对她的感受

例：我妈 50多岁 心疼儿子 经常唠叨 但最关心我
```

#### 如果是导师：
```
用一句话描述你们的关系——她教过你什么、相交多久、教学风格、职位

例：大学数学教授 教了我3年 严谨但很有耐心 对学生要求严格
```

#### 如果是同事：
```
用一句话描述你们的工作关系——做过什么项目、相交多久、她的职位、合作方式

例：产品团队的 UX 设计师 合作2年 思维敏捷 经常有创意碰撞
```

### Q3：性格画像【动态问卷，根据角色类型调整】

#### 如果是前任：
```
用一句话描述她的性格——MBTI、星座、恋爱特点、依恋类型、你对她的印象

例：ENFP 双子座 焦虑型 爱撒娇 翻旧账 嘴上说不在意其实比谁都在意
```

#### 如果是朋友：
```
用一句话描述她的性格——她的优点、相处风格、怎么逗你开心、你们的共同点

例：大大咧咧 经常制造惊喜 能听我吐槽 两个都喜欢看电影
```

#### 如果是家人：
```
用一句话描述她的性格——怎么表达关心、是否啰嗦、你对她的感受、她的价值观

例：虽然会唠叨但心疼儿子 特别想要孙子 坚持原则很固执
```

#### 如果是导师：
```
用一句话描述她的特点——教学特色、与学生的关系、专业精神、你受到的影响

例：讲课很生动 鼓励学生提问 对学术严谨 让我爱上了这个领域
```

#### 如果是同事：
```
用一句话描述她的特点——工作风格、沟通方式、专业特长、相处感受

例：执行力强 会清晰表达想法 对设计细节极度关注 合作很愉快
```

## 字段解析规则

### 识别 MBTI

标准 16 种类型：ISTJ、ISFJ、INFJ、INTJ、ISTP、ISFP、INFP、INTP、ESTP、ESFP、ENFP、ENTP、ESTJ、ESFJ、ENFJ、ENTJ

### 识别星座

12 个星座，包括各种表述方式

### 识别依恋类型

- 安全型
- 焦虑型
- 回避型
- 混乱型

### 识别恋爱性格标签（针对前任）

**常见标签**：
- 爱撒娇、冷暴力、翻旧账、黏人、独立、细腻敏感、忽冷忽热、作、玻璃心、控制欲强
- 直性子、腼腆、外向、内向、易激动、很理性、爱哭、很坚强
- 物质、重感情、很现实、特别浪漫、不懂浪漫、爱吃醋
