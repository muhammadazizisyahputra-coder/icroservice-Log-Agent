# Memories 分析 Prompt

## 任务

你将收到 **{name}** 的原材料（聊天记录、照片、社交媒体等）。
根据 **role_type**，从中提取对应的共同记忆与关系动态。

**原则**：
- 只提取有据可查的内容，不推断
- 没有依据就标注"原材料不足"
- 对感情细节保持温柔与尊重
- **不同角色类型有不同的提取维度**

---

## 提取维度（根据 role_type 动态调整）

### 通用维度（所有角色类型都需要）

#### 1. 关系时间线

从原材料中识别关键节点：
- 重要日期（第一次见面、开始/结束日期、重要纪念日）
- 里程碑（关系升��、重要旅行、重要决定）
- 转折点（冲突、改变、重新开始）

```
输出格式：
重要日期：[日期 + 事件描述]
里程碑：[列表]
转折点：[列表]
```

---

### 前任专用维度

#### 2. 浪漫与亲密

- 约会方式（常去的地点、约会风格）
- 亲密表达（怎么表达爱、身体接触习惯）
- 特殊时刻（提议、第一次说我爱你、同居等）

#### 3. 吵架与冲突

- 常见导火索（什么容易引发争执）
- 升级方式（怎么从不开心变成大吵）
- 冷战特点（冷战多久、怎么修复）
- 从前纠纷（曾经的大矛盾）

#### 4. 思念与想念

- 表达方式（怎么说想你、怎么主动联系）
- 频率（多久想一次、是否经常提起）
- 特定信号（什么时候最容易想起）

---

### 朋友专用维度

#### 2. 共同经历

- 认识过程（怎么成为朋友的、第一印象）
- 一起做过的事（旅行、聚会、冒险）
- 见证过的时刻（对方的重要时刻）

#### 3. 相处风格

- 聊天习惯（怎么聊天、高频话题）
- 开玩笑方式（段子、吐槽、互相损）
- 支持方式（怎么鼓励、怎么陪伴）

#### 4. 共同兴趣

- 喜欢的活动（看什么、玩什么、吃什么）
- 共同价值观（怎么看待同样的事）
- 特殊仪式（定期聚会、每年必做的事）n

---

### 家人专用维度

#### 2. 成长与教养

- 她怎么养大你的（童年回忆）
- 重要教诲（她最常说的话、教过你什么）
- 价值观传递（她坚持什么、怎么影响你）

#### 3. 日常相处

- 日常习惯（她每天都做什么、怎么过日子）
- 家里仪式（家里的规则、节日怎么过）
- 她的脾气（在什么时候最温柔、最严肃）

#### 4. 她对你的关心

- 表达方式（怎么说爱你、怎么照顾）
- 担忧信号（什么时候最紧张、最唠叨）
- 骄傲时刻（你做什么时候她最自豪）n

---

### 导师专用维度

#### 2. 教学内容

- 核心知识点（教过你最重要的什么）
- 教学方法（她怎么讲解、用什么方式教）
- 你学到的最重要的课（概念、方法、精神）

#### 3. 指导时刻

- 印象深刻的课堂（什么课最难忘）
- 关键指导（什么时候点拨了你）
- 师生互动（她怎么鼓励学生、对学生什么态度）n

#### 4. 影响与改变

- 怎么改变了你（思维、选择、人生观）
- 你因她而坚持的东西（兴趣、价值观）
- 她说过的经典话（你记住的金句）

---

### 同事专用维度

#### 2. 工作项目

- 一起做过的项目（项目名、时间、结果）
- 她在项目中的角色（职责、贡献）
- 合作亮点（怎么配合、有什么成就）n

#### 3. 工作风格

- 工作方法（怎么组织工作、做事流程）
- 沟通方式（怎么表达想法、接受反馈）
- 专业能力（擅长什么、深度如何）

#### 4. 办公室关系

- 办公室轶事（趣事、冲突、欢笑时刻）
- 相处感受（是否好相处、是否靠谱）
- 职业发展（她的升职、成就、变化）

---

## 注意事项

1. **原材料不足**：如果某个维度信息不足，用"（暂无足够信息，建议追加聊天记录）"占位

2. **直接引用**：引用她说过的原话时加引号，标注来源日期

3. **避免推断**：不要猜测她的想法，只记录可观察的行为

4. **时间敏感**：标注什么时候发生，便于后续参考时间线
